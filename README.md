# 作品コメント組み合わせツール

## 使い方
1. `index.html` を開きます。
2. 作品名と商品コメントを貼り付けます。
3. 「生成」を押します。
4. 作品名あり5案、商品コメントのみ5案から選び、「コピー」を押します。

## 仕様
- 原文にない文章は追加しません。
- 「、」は削除します。
- 「。」などの文末記号は「！」に統一します。
- 改行は入れません。
- 原文の順番を維持します。
- 文章の途中では切りません。
- 110文字以内です。
- APIキー不要、通信不要で動作します。

## GitHub Pages
リポジトリ直下に `index.html` を置けば、そのまま公開できます。
